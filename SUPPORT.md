# Support

## Getting Help

- **Documentation**: https://docs.forgebuild.app
- **Issue Tracker**: https://github.com/forgebuild/forge/issues
- **Discord**: https://discord.gg/TVmaw9354z

## Before Opening an Issue

1. Check the documentation for your topic.
2. Search existing issues for similar problems.
3. Run `npm run lint` and `npm run typecheck` to verify your build is valid.
4. Include your Forge version from the About dialog.

## Reporting Bugs

Open an issue at https://github.com/forgebuild/forge/issues with:

- Steps to reproduce
- Expected behaviour
- Actual behaviour
- Screenshots (if applicable)
- Forge version and platform
